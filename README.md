## This is the Git Tutorial
To initialize the git
```
git init
```
To see the status of repo
```
git status
```
To add all the file
```
git add .
```
Commit the chnages
```
git commit -m "commit chnage"
```
push the file in github repo
```
git push -u origin main
```
Create a new Branch
```
git branch developer1
```
